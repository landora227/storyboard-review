### 第2步：配置 Firebase（5分钟）

#### 2.1 创建项目
1. 访问 https://console.firebase.google.com/
2. 点击"添加项目"
3. 项目名称：`storyboard-review`
4. 完成创建

#### 2.2 启用数据库
1. 左侧菜单："Realtime Database"
2. 点击"创建数据库"
3. 选择区域：`us-central1` 或 `asia-southeast1`
4. 安全规则："测试模式"

#### 2.3 获取配置
1. 项目设置 → "你的应用"
2. 点击 `</>` 图标
3. 复制 `firebaseConfig` 对象

#### 2.4 填写配置
编辑 `firebase-config.js`，替换这些值：
```javascript
apiKey: "AIzaSy..." ← 你的真实 API Key
authDomain: "your-project.firebaseapp.com"
databaseURL: "https://your-project.firebaseio.com"
projectId: "your-project"
...
```

**⚠️ 必须填写真实值，不能是 YOUR_API_KEY！**

## 🆘 遇到问题？

### 快速诊断

**问题：分享按钮没反应**
- 检查：Firebase 配置是否填写
- 查看：浏览器控制台（F12）错误信息
- 文档：部署说明.md → 故障排除

**问题：实时同步不工作**
- 检查：网络连接
- 检查：Firebase 数据库是否启用
- 查看：控制台错误信息

**问题：页面显示错误**
- 检查：文件是否完整复制
- 检查：HTML 文件名是否正确
- 清除：浏览器缓存后刷新

**问题：Git 推送失败**
- 尝试：先 `git pull` 再 `git push`
- 检查：是否有权限
- 查看：错误信息

### 详细文档
- 📖 部署说明.md → 故障排除章节
- 📖 📋 最终交付清单.md → 常见问题

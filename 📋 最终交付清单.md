## 🆘 故障排除

### 问题1：点击分享按钮没反应
**可能原因**：
- Firebase 配置未填写或填写错误
- Firebase Realtime Database 未启用

**解决方法**：
1. 检查 `firebase-config.js` 配置
2. 打开浏览器开发者工具（F12）查看错误信息
3. 确认 Firebase Realtime Database 已启用

---

### 问题2：分享链接打不开或显示空白
**可能原因**：
- 数据库规则过于严格
- 房间数据未成功保存

**解决方法**：
1. 设置数据库规则为"测试模式"
2. 检查浏览器控制台错误信息
3. 尝试重新生成链接

---

### 问题3：实时同步不工作
**可能原因**：
- 网络连接问题
- Firebase 配置错误

**解决方法**：
1. 检查网络连接
2. 刷新页面重试
3. 查看控制台报错信息

---

### 问题4：下载的 JSON 文件太大
**可能原因**：
- PDF 文件过大
- 参考图过多或未压缩

**解决方法**：
1. 使用 PDF 压缩工具压缩 PDF
2. 压缩参考图后上传
3. 删除不需要的参考图

---

### 问题5：JSON 导入失败
**可能原因**：
- JSON 文件损坏
- 浏览器内存不足

**解决方法**：
1. 重新下载备份
2. 尝试在其他浏览器导入
3. 检查 JSON 文件格式是否正确
